Keep it current (one-time setup)

A

Download the updater script

Save skill-updater.py from the Skill Builder project to your Mac — put it anywhere permanent, e.g. ~/scripts/skill-updater.py.

B

Install the weekly scheduler (run once in Terminal)

python3 ~/scripts/skill-updater.py --install

Runs every Monday at 9 AM automatically. All future skills you build with recurring sources are picked up with no extra setup.

C

Run it manually anytime

python3 ~/scripts/skill-updater.py

Shows a summary of new content, asks you to accept or skip before updating. Type a to always-accept a source and never be asked again.
